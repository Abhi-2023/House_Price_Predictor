
# House_Price_Predictor
ML tool to predict the house price as per stated given data.

# Software And Tools Required
1. [Github Account](https://github.com)
2. [Heroku Account](https://heroku.com)
3. [VS Code Editor](https://code.visualstudio.com)


# Create new environment

```
conda create -p venv python=3.10 -y
```
Now activate the environment 
```
conda activate .\venv\
```

# Install required Libraries

```
pip install <lib names>
```

# Add requirements.txt file
Create the requirements.txt file under root folder

Add the libraries installed in requirements.txt

```
pip freeze > requirements.txt
```
